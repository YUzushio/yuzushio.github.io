# Gallery

Public portfolio site (SPA) built with Vite + TypeScript.

> 個人の制作・実績をまとめたポートフォリオです。

## Related

## Development

```bash
npm install
npm run dev
```

## Deploy

GitHub Pages is deployed automatically on push to `main` via GitHub Actions.

After the first deploy, enable **Settings → Pages → Build and deployment → GitHub Actions**.

Site URL: `https://<username>.github.io/gallery/`
